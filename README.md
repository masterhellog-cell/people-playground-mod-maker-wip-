# People Playground Mod Maker AI

A web app that uses AI to generate People Playground mods!

## Setup Instructions

1. Make sure you have Node.js installed (download from https://nodejs.org/)
2. Open Command Prompt (cmd) or PowerShell in this folder
3. Run these commands:
   ```bash
   npm install
   node server.js
   ```
4. Open your browser and go to http://localhost:8000
5. Describe your mod and click "Generate Mod"!

## Files
- index.html: Main UI
- styles.css: Styling
- app.js: Application logic (AI integration)
- server.js: Local server to handle CORS
- package.json: Dependencies
- start.bat: Quick start (Windows only)

## Notes
- Uses opencode.ai API
- Generates script.cs and mod.json
- Downloads as a ZIP file
